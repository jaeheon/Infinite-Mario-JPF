Infinite-Mario-Mac by Lewisham

A Mac OS X compatible version of Infinite Mario

Infinite Mario, as provided by the original author, does not function on Mac OS X due to a timing bug. This version removes the author's homebrew timing to use Java System's getMillis instead.

Dependencies

You need Maven 2 to build this, and then it will fetch the required packages for you.

Install

mvn package Look for a file like "Mario-0.0.1-SNAPSHOT.jar" in the target/ folder.
